## 如果 Railway 部署失败

检查 Railway 项目设置：
- Build Command: `pip install -r requirements.txt`
- Start Command: `gunicorn app:app --bind 0.0.0.0:$PORT --workers 1 --timeout 300`

如果 Railway 提示找不到模块，在 Railway 的 Variables 里添加：
```
PYTHON_VERSION=3.10
```
