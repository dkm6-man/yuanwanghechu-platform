## 第二步：Railway部署

1. 打开 https://railway.app
2. 点击 "Start a New Project"
3. 选择 "Deploy from GitHub repo"
4. 用GitHub账号登录授权
5. 选择 `yuanwanghechu-platform` 仓库
6. Railway 会自动检测 Python 项目并部署
7. 等待 2-3 分钟，部署完成后会显示域名如:
   ```
   yuanwanghechu-platform.up.railway.app
   ```
