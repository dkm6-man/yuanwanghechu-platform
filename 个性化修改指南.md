### 修改每个元素的颜色
**文件:** `static/css/style.css` (约第218-250行)

```css
/* 电力（金色） - 改这些颜色值 */
.el-power {
    background: radial-gradient(circle, rgba(255, 215, 0, 0.3), rgba(255, 152, 0, 0.1));
    /*               ↑ 改这两个RGBA值改背景色 */
    border: 2px solid rgba(255, 215, 0, 0.5);
    /*                ↑ 改边框颜色 */
    box-shadow: 0 0 20px rgba(255, 215, 0, 0.3);
    /*                  ↑ 改发光颜色 */
}

/* 推荐的配色方案：
   金色系: rgba(255, 215, 0, ...)   - 电力 ⚡
   蓝色系: rgba(0, 200, 255, ...)   - 算力 💻
   红色系: rgba(255, 87, 34, ...)   - 负荷 🏭→🌡️
   绿色系: rgba(76, 175, 80, ...)   - 储能 🔋
*/
```

## 2️⃣ 修改整体配色

### CSS变量（颜色总控）
**文件:** `static/css/style.css` (第8-20行)

```css
:root {
    --primary: #1a73e8;        /* 主色调（按钮、链接） */
    --primary-dark: #1557b0;   /* 主色调深色 */
    --accent: #ff6d00;         /* 强调色（注册按钮等） */
    --bg-dark: #0a0e27;        /* 深色背景（导航栏、首页） */
    --bg-card: #ffffff;        /* 卡片背景 */
    --text: #333333;           /* 正文颜色 */
    --text-light: #666666;     /* 浅色文字 */
    --success: #2e7d32;        /* 成功绿色 */
    --error: #c62828;          /* 错误红色 */
    --radius: 12px;            /* 全局圆角 */
}
/* 只改这里，全局颜色就变了 */
```

## 🎯 快速修改清单

| 想改什么 | 改哪个文件 | 怎么改 |
|---------|-----------|--------|
| 4个元素文字/emoji | `index.html` | 直接改HTML文字 |
| 元素颜色 | `style.css` | 改 `.el-power` 等的颜色值 |
| 元素大小 | `style.css` | 改 `.orbit-element` 的 width/height |
| 旋转速度默认值 | `index.html` | 改滑块 `value="60"` |
| 速度范围 | `index.html` | 改滑块 `min/max` |
| 整体配色 | `style.css` | 改 `:root` 里的CSS变量 |
| 标题/描述文字 | `index.html` | 直接改文字 |
| 背景渐变 | `style.css` | 改 `.index-page` 的 gradient |
| 功能卡片 | `index.html` | 改卡片里的文字 |
| 参数默认值 | `compute_s1.html` 等 | 改 `value=` 属性 |
| 用户协议 | `agreement.html` | 直接改条款 |
| 管理员密码 | 删 `data.db` 重运行 | 或直接改注册逻辑 |

## 💡 技巧
- 修改后用浏览器 `Ctrl+F5` 强制刷新看效果
- CSS改颜色支持：十六进制(#ff0000)、RGB、RGBA
- Emoji可以去 https://emojipedia.org 复制
- 每个 `.el-xxx` 独立控制一个元素的样式
